## Recipe Finder
- An application for finding food recipes

## Programmer
- Heang Ly

## Specifications:
- Allows users to type in any name of food
- Displaying 12 recipes that are related to the input word
- Allows users to vote their favorite recipes
- Displaying top 5 recipes that have the most votes

## Technologies
- ReactJs
- React Router
- NodeJs
- Express
- Node-Postgres
- UUID
- Font Awesome
